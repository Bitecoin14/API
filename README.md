# API

A RESTful API project.

## Getting Started

### Prerequisites

- Node.js (or your preferred runtime)
- Package manager (npm, yarn, etc.)

### Installation

```bash
git clone https://github.com/bitecoin14/api.git
cd api
npm install
```

### Running the Server

```bash
npm start
```

## Usage

Base URL: `http://localhost:3000`

### Endpoints

| Method | Path | Description |
|--------|------|-------------|
| GET    | `/`  | Health check |

## Configuration

Copy `.env.example` to `.env` and fill in the required values.

```bash
cp .env.example .env
```
